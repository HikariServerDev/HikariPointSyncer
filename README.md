# HikariPointSyncer - 共有ウェイポイント同期システム

**HikariPointSyncer** は、マルチプレイヤーサーバー内の全プレイヤー間で **Xaero's Minimap** および **Xaero's World Map** のウェイポイントをリアルタイムに共有・同期・管理するためのマインクラフト（Fabric 1.18.2）Modです。

ゲーム内のHUDやマップ画面の見た目の美しさを崩すことなく、完璧にバックグラウンドで座標情報の自動同期を行います。

---

## 主な機能

*   **簡単アップロード**: 
    ローカル（あなたのXaero's Minimap内）に作成したウェイポイントを、ワンクリックでサーバーの全員に向けて共有リストへアップロードできます。
*   **重複アップロード防止**: 
    すでに共有リストにアップロードされている（またはダウンロード済みの）ウェイポイントは、ローカル一覧で「済み」とブラックアウトされ、二重に共有してしまうのを防ぎます。
*   **ワンクリックダウンロード**: 
    他のプレイヤーが共有したウェイポイントをワンクリックでダウンロードし、自分のマイクラに登録できます。
*   **リアルタイム強制削除同期**: 
    自分が共有したウェイポイント、または**OP権限（サーバー管理者）**を持つプレイヤーが共有リストからウェイポイントを削除すると、**現在サーバーにいる全プレイヤーの画面やファイル上から対象のウェイポイントが即座に強制抹消**されます。オフラインだったプレイヤーも、次回ログイン時に自動クリーンアップされます。
*   **ごみ箱（復元機能）と30日間保持**: 
    削除されたウェイポイントは「削除したWaypoint」リストに移動し、30日間保管されます。30日以内であれば作成者または管理者がいつでも復元可能です。残り日数はGUI上にカウントダウンでリアルタイムに表示されます。
*   **スマート自動更新**: 
    新しいバージョンがリリースされた場合、起動時に通知ダイアログが表示されます。「更新する」を押すだけで自動で過去のバージョンを完全消去し、最新版をダウンロードします。

---

## 🛠️ 対応仕様

*   **対応Minecraftバージョン**: Minecraft Java Fabric v1.18.2
*   **対応OS/ランチャー環境**: すべてのOS、およびランチャーで動作します
*   **前提Mod**: 
    *   Fabric Loader / Fabric API
    *   Xaero's Minimap または Xaero's World Map

## 使用方法

### 1. メイン画面を開く
ゲーム内で **`Right Control` (右側のコントロールキー)** を押すと、同期メニュー画面が開きます。
※コントロール設定からお好みのキーに変更することも可能です。

### 2. ウェイポイントをアップロードする
1.  メイン画面の右下にある **「ローカルから共有する」** ボタンを押します。
2.  表示されたローカルウェイポイント一覧から、共有したいウェイポイントの右側にある **「アップロード」** ボタンを押します。
3.  アップロードが成功すると、ボタンが **「済み」** に切り替わり、サーバー内の全員の共有リストにリアルタイムで追加されます。

### 3. 他の人のウェイポイントをダウンロードする
1.  メイン画面の **「共有中リスト」** タブを開きます。
2.  ダウンロードしたいウェイポイントの右側にある **「DL」** ボタンを押します。
3.  ダウンロードが完了すると、自動的に Xaero's Minimap のシステムに登録され、あなたのゲーム画面やミニマップ上に対象のウェイポイントが表示されます。

### 4. ウェイポイントを削除・復元する
*   **削除**: 
    「共有中リスト」で、自分がアップロードした（または自分が管理者である）ウェイポイントの右端にある **「削除」** ボタンを押すと、共有リストから削除され、全プレイヤーのゲーム画面から即時に強制削除されます。
*   **復元**: 
    削除されたウェイポイントは **「削除したWaypoint」** タブに移動します。30日以内であれば、右端の **「復元」** ボタンを押すことで元の共有中リストにいつでも戻すことができます。

## 自動アップデートの実行方法

1.  Modの新しいバージョンがリリースされている場合、マイクラ起動時に自動的に **Update画面** が表示されます。
2.  **「更新する」** を押すと、バックグラウンドで自動的に最新のJARファイルがダウンロードされ、同時に `mods` フォルダ内にある古いJARファイルが自動的に消去されます。
3.  ダウンロードが完了すると、画面に「ダウンロードが完了しました。ランチャーを再起動してください。」と表示され、ボタンが以下に切り替わります：
    *   **「再起動する」**: マインクラフトを安全に終了させ、自動でランチャーを通じてマイクラを再起動します。
    *   **「再起動しない」**: ゲームのタイトル画面へ戻ります（次回のゲーム手動起動時に更新が適用されます）。
